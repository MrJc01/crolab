# Estudo Competitivo: DigitalOcean / Paperspace Legacy

## 1. Arquitetura (Droplets Clássicos e UX)
A DO foi o pilar máximo de "Developers First" antes das IA explodirem. Eles forjaram o "Droplet" (uma VM instantânea baseada em KVM com IP exposto em Segundos). Hoje, compraram a Paperspace para brigar com a CoreWeave.

## 2. Média de Valor/Custo
Geralmente presevam a mentalidade transparente sem os horrores de taxa reversa (Egress). VMs com placas H100 giram em US$`~ 3.00/h` até T4 em `~$0.50/h`.

## 3. O Fator UX da Digital Ocean
A mágica da DO é o Design do Dashboard. É Limpo, rápido e objetivo. Diferente da Amazon onde criar uma nuvem gasta 8 clicks e 3 caixas de opções infinitas.

## 4. Insight Crolab (O Design Ideal)
Se formos criar o Webapp Crolab Dashboard para vender orquestração das instâncias do vast.ai, a Linguagem Visual de Design deve emular brutalmente o "Minimalismo Elegante" da DigitalOcean. Quando o App do Crom for escrito, ele será de Vidro (Glassmorphism abstrato) guiando o usuário focado nas "GPUs Node Agentes".
